# Sales Forecasting Project

A practical sales forecasting project using Python, notebooks, and generated output artifacts.

## Project Structure

- `dmart_sales.py`: main Python script for data processing and forecasting workflow.
- `DMart_Sales_Forecasting_FIXED.ipynb`: primary notebook version.
- `datasets/`: source and processed dataset files.
- `outputs_sales_forecasting/`: forecast outputs, reports, and model artifacts.

## Requirements

- Python 3.10+ recommended
- Common libraries used in this project include:
  - `pandas`
  - `numpy`
  - `scikit-learn`
  - `matplotlib`
  - `prophet` (if running Prophet models)

## How To Run

1. Create and activate a virtual environment.
2. Install dependencies.
3. Run:

```bash
python dmart_sales.py
```

Or open and run cells in:

```bash
DMart_Sales_Forecasting_FIXED.ipynb
```

## Notes

- Some model/output files can be large. GitHub blocks files larger than 100MB unless Git LFS is used.
- This repository is organized as one-file-per-commit history for easier tracking.
